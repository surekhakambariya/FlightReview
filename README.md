## OpenFlights 
### A flight reviews app built with Ruby on Rails and React.js

This app is intended to be a simple example of a CRUD app built with **Ruby on Rails** and **React.js** using **Webpacker**.

This app uses:

* Ruby version: `3.0.1`
* Rails version: `7.0.0`
* Database: `postgresql`
* React version: `16.12.0`
* React Hooks API
* React Context API
